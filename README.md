# TwoThousandYearOldVampire
A legally distinct version of TYOV integrated for Evil Neuropathy Samantha Vedal to play

Setup for the NeuroSDK:
* Open the game.
* Open the save file at C:\Users\ username here \AppData\Roaming\Godot\app_userdata\Two thousand year old vampire\
* Change the Websocket var to the address the SDK is using
* Reload the game and Evil will start playing
